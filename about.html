About The Project

Architecting modern travel web experiences with clean engineering & stunning UI/UX

Welcome to TravelGo, your ultimate luxury travel companion! Developed and engineered by Nidhi Mehta, this frontend web application is designed to simplify and elevate the vacation planning journey.

Featuring real-time dynamic search, category filtering, interactive day-by-day itinerary previews, state-driven price calculation, and seamless dark mode theming, TravelGo reflects state-of-the-art web development standards.

⚡ Pure JavaScript ES6+ 🎨 Modern CSS Variables 📱 Responsive UI/UX ✨ Glassmorphism
